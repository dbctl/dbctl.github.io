Welcome to dbctl - the tool designed to make spin up a local database in seconds

Features

  • Run Databases with Applied Migrations: Run a database with applied migration files and sample data.
  • Access Connection URL: Access the connection URL and use your preferred tools to connect to the database.
  • Launch User Interface: Launch a user interface (UI) for managing the database, if available.
  • Quick Tests: Enable users to quickly run tests in a fresh database.
  • Easy Cleanup: Cleanup databases once you're finished using them!

About dbctl

dbctl is a tool designed to start a database in seconds. It offers features like running databases with applied migration files and sample data, accessing connection URLs, launching user interfaces for managing the database, quick tests, and easy cleanup. dbctl is not intended for running databases in production. Its primary purpose is to simplify testing and practice with no hassle.
